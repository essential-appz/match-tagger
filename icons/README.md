# Icons Directory

This directory should contain PWA icon files for Match Tagger.

## Required Icons

- `icon-192x192.png` - 192x192 pixel icon
- `icon-512x512.png` - 512x512 pixel icon

## Creating Icons

You can:
1. Copy icons from player-tagger if you want similar branding
2. Create new icons with a design tool
3. Use an online icon generator
4. Use placeholder icons for testing

## Temporary Solution

For testing, you can copy icons from player-tagger:
```bash
cp ../player-tagger/icons/icon-192x192.png ./icon-192x192.png
cp ../player-tagger/icons/icon-512x512.png ./icon-512x512.png
```

Or create simple colored placeholder PNGs with any image editor.
